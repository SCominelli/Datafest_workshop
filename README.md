# Datafest_workshop
Undergraduate R Workshop: Weather Data Analysis
A comprehensive R script demonstrating tidy data principles and statistical foundations. 

**Tools**: R, RStudio, Tidyverse (dplyr, ggplot2, lubridate). 

**Concepts**: Data cleaning pipes (%>%), unit conversion, seasonal trend visualization, and basic linear modeling (lm). 

**Goal**: To move beyond memorizing code toward thinking critically about data organization and interpretation.
